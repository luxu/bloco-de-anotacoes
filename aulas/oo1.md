# Dicas do Terminal
Vou listar alguns comando que estamos aprendendo:

- `cd` (navega entre pastas)
Exemplo para entrar dentro de uma pasta
```
cd nomeDaPasta
```
Exemplo para sair de uma pasta
```
cd ..
```

- `mkdir` (cria pastas)
Exemplo de como criar uma pastas
```
mkdir nomeDaPasta
```
